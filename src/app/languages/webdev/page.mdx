# Introduction to Web Development

Frontend web development involves creating the part of a website that users interact with. It consists of three core technologies:

- **HTML (HyperText Markup Language)**: Defines the structure of a webpage.
- **CSS (Cascading Style Sheets)**: Styles the webpage to make it visually appealing.
- **JavaScript**: Adds interactivity to the webpage.

---

## HTML Basics

HTML provides the basic structure of a webpage using elements enclosed in tags.

### Example:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Welcome to My Website</h1>
    <p>This is a paragraph.</p>
    <a href="https://www.example.com">Click here</a>
  </body>
</html>
```

### Key Elements:

- `<h1> - <h6>`: Headings
- `<p>`: Paragraph
- `<a href="URL">`: Link
- `<img src="image.jpg">`: Image
- `<ul>, <ol>, <li>`: Lists

---

## CSS Basics

CSS is used to style HTML elements to make the webpage visually appealing.

### Example:

```css
body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}
h1 {
  color: darkblue;
  text-align: center;
}
```

### Key Concepts:

- **Selectors**: Target HTML elements (`h1`, `p`, `div`)
- **Properties**: Define styles (e.g., `color`, `font-size`)
- **Values**: Define property values (e.g., `red`, `20px`)

---

## JavaScript Basics

JavaScript makes webpages interactive.

### Example:

```html
<button onclick="sayHello()">Click Me</button>
<script>
  function sayHello() {
    alert("Hello, World!");
  }
</script>
```

### Key Concepts:

- **Variables**: Store data (`let x = 10;`)
- **Functions**: Perform actions (`function greet() {}`)
- **Events**: Respond to user actions (`onclick`, `onmouseover`)

---

## Roadmap for Learning Frontend Development

### Step 1: Learn the Basics

- HTML: Structure of webpages
- CSS: Styling techniques
- JavaScript: Basic interactivity

### Step 2: Intermediate Skills

- CSS Flexbox & Grid: Layout design
- JavaScript DOM Manipulation: Dynamic content updates
- Responsive Design: Making sites mobile-friendly

### Step 3: Advanced Topics

- CSS Frameworks: Bootstrap, Tailwind CSS
- JavaScript ES6+: Modern JS features
- APIs & Fetch: Data handling

### Step 4: Build Projects

- Personal Portfolio
- To-Do List App
- Weather App (using API)

### Step 5: Learn a Framework

- **React.js**: Popular library for UI development
- **Vue.js**: Lightweight and beginner-friendly alternative

### Step 6: Deploy Your Work

- Git & GitHub: Version control
- Netlify/Vercel: Free hosting for frontend projects

By following this roadmap and practicing regularly, you'll become a skilled frontend developer!
